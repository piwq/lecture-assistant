# Лекция-ассистент 🎓

Веб-приложение для транскрибации лекций в реальном времени и автоматической генерации ответов на вопросы преподавателя с использованием моделей Google Gemini.

## Структура проекта
* `server.py` — backend на Flask (отвечает за многопоточные запросы к API Gemini и сохранение в `.docx`).
* `lecture.html` — frontend-интерфейс (распознавание речи через Web Speech API, управление аудио).
* `requirements.txt` — список необходимых Python-библиотек.
* `lecture.docx` — файл, в который автоматически сохраняются сгенерированные ответы (создается при первом запуске).

## 🚀 Установка и запуск на Windows

### Шаг 1: Подготовка
Убедитесь, что на компьютере установлен **Python** (при установке обязательно поставьте галочку *"Add Python to PATH"*).

### Шаг 2: Настройка окружения
Откройте терминал (PowerShell или `cmd`) в папке с проектом и выполните следующие команды:

1. **Создайте виртуальное окружение:**
   ```cmd
   python -m venv venv
   ```

2. **Активируйте виртуальное окружение:**
   ```cmd
   venv\Scripts\activate
   ```
   *(Успешно активированное окружение отобразит префикс `(venv)` в начале строки терминала).*

3. **Установите зависимости:**
   ```cmd
   pip install -r requirements.txt
   ```

### Шаг 3: Запуск сервера
При активированном окружении запустите главный скрипт:
   ```cmd
   python server.py
   ```

После успешного запуска откройте браузер **Google Chrome** (важно для поддержки Web Speech API) и перейдите по адресу:  
👉 **http://localhost:3333**

## 🛠 Возможные проблемы (Troubleshooting)

**Ошибка активации `venv` в PowerShell:** Если при вводе команды `venv\Scripts\activate` появляется красный текст с ошибкой "выполнение сценариев отключено" (Execution Policies), сделайте следующее:
1. Откройте PowerShell от имени Администратора.
2. Введите команду: `Set-ExecutionPolicy RemoteSigned`
3. Нажмите `Y` и `Enter`.
4. Вернитесь в терминал проекта и повторите попытку активации.
